## Setup & Requirements
To run the Jupyter Notebook successfully:
1. Ensure you have **PyTorch** and **torchvision** installed.
2. The notebook is configured to run in a Kaggle environment and looks for the dataset at: `/kaggle/input/datasets/akash2sharma/tiny-imagenet/tiny-imagenet-200`. If you are running locally, please update the `DATASET_PATH` variable in the notebook to point to your local path of the downloaded Tiny-ImageNet-200 dataset.
